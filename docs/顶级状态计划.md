## 北极星：三个体验判据

1. **跟手**：从按下鼠标到画面响应，中间没有任何一次网络、磁盘、导出、握手（宪法，已达成，永不许破）。
2. **可预期**：每个动作的结果和 Figma/Miro 的直觉一致；每次点击必有可感知回应；出错自愈，永不要求用户"刷新/重试/稍候"。
3. **一体**：卡片、墨迹、便签、终端框是同一个世界——同一相机、同一撤销链、同一选择语义、同一视觉语言。

## 现状底座（不需要重做的部分）

- 单一场景文档 + 乐观优先 + 后台冲刷（scene-store）；服务端 LWW 快照仓；全画布 Cmd+Z。
- 自研墨迹三件套（ink.js / InkLayer / InkTools）：笔/矩形/椭圆/箭头/文字，落笔即文档、直渲 SVG、单相机。
- 验收：node 65/65、4518 原生八链、300 元素直渲 649ms、构建 75ms。
- 终端框（倒序分页）、交接三件套（bingo-agent-handoff 桥接）、launchd 常驻、每日备份。

## 差距清单（分组给现状与顶级标准，怎么达成由你定）

### A · 墨迹手感补全（哥最在意）
- **选中操控**：现状只有移动/删除。顶级标准：选中后有 resize 手柄（形状/文字/图片）、可旋转；操作全程跟手、全程一步 undo。
- **多选**：现状单选。顶级标准：选择工具框选多元素、批量移动/删除/改样式。
- **复制粘贴**：现状无。顶级标准：Cmd+C/V 复制选中墨迹（含跨会话粘贴进同画布）、Alt+拖=复制移动。
- **图片**：现状只渲染存量。顶级标准：粘贴/拖入图片文件即成元素（走既有 drawingFiles 内容寻址仓）。
- **文字体验**：现状就地 textarea 够用。顶级标准：双击任意文字即编辑、字号可调、编辑框随缩放不糊。
- **橡皮/快捷键**：V 选择、P 画笔、R 矩形、O 椭圆、A 箭头、T 文字、Esc 收工具——和 Figma 肌肉记忆对齐。

### B · 整理与布局
- 现状：整理落定 520ms 节点滑入，但墨迹瞬移。顶级标准：墨迹与所属容器同曲线滑动，整场如一次呼吸。
- 整理语义：阅读顺序自然（活跃者靠前）、间距有节奏、画板参与排布不撞车；整理前后哥能一眼说出"变整齐了"。

### C · 全局手感
- 拖工作区/便签/容器在 352 节点规模下全程 60fps（Performance 面板证据）。
- 悬停、光标、选中态、拖放高亮的每一处微交互过一遍 Figma 级审美（apple-design 的克制标准）。
- 同步状态小点（saved/dirty/error）在 TopBar 可见但不聒噪。

### D · 稳定性
- 多标签同开：LWW 语义下不打架、不弹任何"请刷新"。
- 断网/daemon 重启期间：输入照常、角落亮"未同步"、恢复后自动追平（现有退避机制验证到位即可）。
- pagehide 冲刷实测可靠（关标签→重开零丢失）。

### E · 验收体系
